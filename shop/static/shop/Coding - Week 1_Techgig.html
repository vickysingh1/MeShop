Problem Statement

Women's IPL (100 Marks)

The popularity of Women’s Cricket is increasing and India being the Cricket Nation has a lot of responsibility to make it as popular as the men’s cricket. Women’s IPL was hosted this year and it was a huge success. The next edition needs to be better and the preparations for the same have already begun. The previous edition of IPL had limited number of players and owners.


For the coming edition, N cricketers have made them available for the IPL. All the cricket boards have allowed their players to play in the IPL. The auction is to be held in few months from now. The Host for the auction has the profile of all N cricketers but is helpless as he has no systematic way of auctioning the players. The profile consists of the Name of the player in lowercase letters (a-z), and the current rating R she has. The rating is calculated on the basis of players performance over the past few years in the T20 format.


The host will reward you with free tickets for the opening ceremony if you could help him provide the names of the players in the decreasing order of their ratings. There are some special cases which neede to be handled:


Case 1: If the rating of two players is same, then their names should be provided on the basis of first come first serve i.e. player whose name appeared first should be put above the other.

Can you help him provide the list of names as required ?

Input Format

The first line of input consists of number of test cases, T.

The first line of each test case consists of number of players, N.

Next N lines of each test case consists of name of player and her rating R space separately.



Constraints

1<= T <=10

1<= N <=100000

1<= |name_of_player| <=50

0<= R <=100000



Output Format
For each test case, print the names of players in correct order with each name being on a separate line.
